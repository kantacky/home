## 俳句の基本原則

### 俳句とは何か

俳句は**十七音の中に一瞬を切り取る詩形**である。説明ではなく、映像と余白によって読み手の中に感覚を立ち上げる。作者の感動を語るのではなく、感動の原因となった景を置くことで、読み手が自ら感じ取る余地を残す。

### いい俳句の条件

1. **一物仕立て、または取り合わせが効いている**
   - 一物仕立て: 一つの対象を深く写生し、そこに発見がある。
   - 取り合わせ: 意味の離れた二つの要素を切れで並べ、響き合わせる。距離が近すぎれば説明、遠すぎれば無関係になる。

2. **季語が一句の核になっている**
   - 季語は飾りではなく、句の世界観を支える柱。一句に季語は一つ。
   - 強い季語（桜、雪、月など）は周辺の説明を削るほど映える。
   - 季語の本意（その語が歴史的に担ってきたイメージ）を踏まえつつ、新しい角度で提示できると強い。
   - 迷う場合は [kigo-and-kireji.md](references/kigo-and-kireji.md) を参照する。

3. **切れがある**
   - 切れは俳句に奥行きを与える構造上の段差。意味の流れを一度断ち、余韻や飛躍を生む。
   - 切れ字（や・かな・けり）を使うか、構文上の断絶で実現する。
   - 切れがなければ散文の断片にすぎない。
   - 切れ字の使い分けは [kigo-and-kireji.md](references/kigo-and-kireji.md) を参照する。

4. **具体的な映像がある**
   - 「見える・聞こえる・触れる・匂う」ものを名詞中心に置く。
   - 抽象語（寂しい、美しい、儚い）は映像に置き換える。感情は景に語らせる。

5. **余白がある**
   - 言い過ぎない。説明しすぎない。全部を語らない。
   - 読み手が想像で補う余地こそが俳句の力。
   - 教訓・標語・感想文にならないよう注意する。

6. **音の調べが自然である**
   - 五七五は原則だが、絶対ではない。声に出したときの流れを優先する。
   - 字余り・字足らずは意図があれば許容されるが、破調には理由が必要。

### 避けるべきこと

- 季語の重複（季重なり）: 意図的でない限り避ける。
- 説明・報告: 「〜だった」「〜と思う」は散文。
- 比喩の過剰: 一句に比喩は一つまで。
- 口語と文語の無自覚な混在。
- 固有名詞の安易な使用: 季語や映像を壊す場合は削る。

## 作句の手順

1. **素材を掴む** — 依頼から「見えるもの・聞こえるもの・触れるもの」を3〜5個抽出する。
2. **季語を一つ決める** — 素材に最も合う季語を選ぶ。
3. **骨格を作る** — 二つの構図で試す。
   - 構図A: 上五=場 / 中七=動き / 下五=余韻
   - 構図B: 上五中七=写生 / 下五=切れ
4. **切れを入れる** — 意味の段差を一か所作る。
5. **推敲する** — [revision-checklist.md](references/revision-checklist.md) に沿って削り、整える。
6. **候補を比較する** — 最低3案を並べ、各案に強み・弱みを付ける。

---

## 出力形式

1. 仕上げた俳句（1句）
2. 別案（2句まで）
3. 改善ポイント（3点以内）
4. 次に試す一手（季語変更・切れ位置変更・語順変更のいずれか）
